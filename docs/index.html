Features

Everything yt-dlp, without the terminal.

Catalyst wraps the best download engine on the internet in a UI that's actually pleasant to use.

1800+ sites

YouTube, Vimeo, Twitter, TikTok, Instagram, Reddit, Twitch, Bilibili — and hundreds more, powered by yt-dlp.

Smart queue

Add URLs in bulk, reorder by dragging, control how many run at once. Active downloads at the top, completed below.

Format control

Choose container, codec, and resolution. MP4 (H264) for compatibility, Best Quality for archive, MP3/M4A for audio.

Native & lightweight

Built with Tauri — uses the OS's native WebView, not Chromium. Tiny binary, low memory, genuinely fast.

Private by design

No analytics. No telemetry. No account. Everything runs locally on your machine. Open source and auditable.

Full history

Every download logged with thumbnail, channel, date, size. Search by title or URL. Re-download with one click.

Output categories

Create named destinations — Movies, Music, Work — each with its own directory and colour. Select when adding downloads.

Auto-updated engine

yt-dlp updates silently in the background so new sites and fixes land without you doing anything.

Cookie support

Pull cookies automatically from Chrome, Edge, Firefox, Brave — or import a file. Unlocks members-only content.

How it works

Paste. Pick. Done.

1

Paste a URL

Drop in any link from YouTube, Vimeo, Twitter, or 1800+ other sites. Bulk import a list from a text file in one shot.

2

Choose format

Select container, quality cap, and output category. Catalyst fetches the title and thumbnail instantly.

3

Watch it arrive

Live progress bars, speed, ETA. When done, the file lands clean in your output folder. Get an OS notification if the window isn't focused.

Start downloading.

Free, open source, always will be.
